# Парсер лотов с Torgi.org

## Описание
Программа парсит информацию о лотах с сайта Torgi.org, извлекает данные о названиях, ценах и ссылках на лоты, сортирует их по убыванию цены и позволяет фильтровать по заданному диапазону цен.

## Источник данных
Данные берутся с официального сайта торгов:
https://torgi.org/index.php?class=Auction&action=List&mod=Open&AuctionType=All

## Установка зависимостей
1. Клонируйте репозиторий:
```bash
git clone https://github.com/KarinaMikalina/torgi_parser.git
cd torgi_parser
```
2. Создайте виртуальное окружение:
```bash
python -m venv venv
```
3. Активируйте виртуальное окружение:
```bash
# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate
```
4. Установите зависимости:
```bash
pip install -r requirements.txt
```
## Запуск программы
1. Скачайте HTML-страницу:
```bash
python download.py
```
2. Запустите парсер:
```bash
python parse_trades.py
```
